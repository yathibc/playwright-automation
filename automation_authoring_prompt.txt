==================================================
LOCATOR STRATEGY REQUIREMENTS
==================================================

Use resilient locator discovery and coding practices.

Prefer this order:
1. accessibility-aware locators (`getByRole`, `getByLabel`)
2. visible text locators with stable text fragments
3. semantic containers and relation-based locators
4. stable attributes and data-* selectors
5. fallback locator chains

Avoid brittle selectors unless there is no alternative.
If fallback selectors are necessary, document why.

For dynamic UIs:
- re-query elements after re-render
- tolerate delayed rendering and overlays
- verify visibility and enabled state before acting
- use retries where appropriate
